# Automatic Metric Results

This directory contains retained automatic metric outputs for the public
five-system comparison and HT reference row described in the preprint.

Key files:

- `all_results.csv`: combined system, book, and metric scores.
- `summary.json`: aggregate summary generated from `all_results.csv`.
- `dev/` and `eval/`: per-book score files for the retained systems.

Text-bearing fields are redacted or withheld in this public branch.
