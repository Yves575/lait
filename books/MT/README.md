# Public Machine-Translation Outputs

This directory contains the public MT outputs for the systems discussed in the
preprint.

| Path | Contents |
| --- | --- |
| `pipeline1/gemini/` | P1 outputs from Gemini. |
| `pipeline1/gpt54_high/` | P1 outputs from GPT-5.4 High. |
| `pipeline2/gemini/` | P2 outputs from Gemini. |
| `pipeline2/gpt54_high/` | P2 outputs from GPT-5.4 High. |
| `pipeline3/` | P3 agentic-pipeline outputs. |

The `pipeline3/extern/` subdirectory contains the multilingual target-language
examples referenced in the appendix.
